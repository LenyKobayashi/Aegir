Aegir
=====

Bonjour,

Bienvenue sur le projet Aegir.

premier pas avec github:

1- creer son premier github
2- une fois son compte github créé il faut se rendre sur https://github.com/elixian/Aegir.git et cliquer sur fork en étant connecté sur son compte.
3- là c'est peut être le pls chiant :
    - sur sa bécane, creer le répertoir de réception ATTENTION si utilisation d'un IDE creer se répertoir dans le Workspace correspondant
    - en ligne de commande se rendre dans ce répertoire et faire $ git clone https://github.com/USERNAME_DE_VOTRE PSEUDO_GIHUB/Aegir.git
4- Si problème on verra ça ensemble

ENJOY !!!
